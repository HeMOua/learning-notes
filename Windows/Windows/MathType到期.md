# MathType到期

## 适用于6.9版本

- **关闭 Word**
- win + R 打开运行窗口，输入regedit
- 按照如下目录找到注册表
  计算机\HKEY_CURRENT_USER\SOFTWARE\Install Options
- 删掉options6.9
