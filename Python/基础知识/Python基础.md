# 基础语法

## 1、import用法

| 方式                        | 引用方式                    |
| --------------------------- | --------------------------- |
| import <库名>               | <库名>.<函数名>(<函数参数>) |
| from <库名> import <函数名> | <函数名>(<函数参数>)        |
| from <库名> import *        | <函数名>(<函数参数>)        |
| import <库名> as 别名       | <别名>.<函数名>(<函数参数>) |
